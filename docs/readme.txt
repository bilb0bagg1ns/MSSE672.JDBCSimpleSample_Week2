
Sample was built using:

mysql-5.7.12
mysql-connector-java-8.0.11.jar

Instructions to run sample:

1. Make sure you are done with MySQL tutorial of installing MySQL and configuring it.

2. Download the JDBC Driver from https://dev.mysql.com/downloads/connector/j/

   Unzip the download and extract and copy JDBC driver jar to the lib folder

4. Compile and Run.
